## Versão Web  
A versão web da aplicação foi desenvolvida em **React** com o **Vite**.

### Comandos para instalar

1. 
   ```bash
   npm create vite@latest autohub-provider -- --template react
   ```

2. 
   ```bash
   cd autohub-provider
   ```

3. 
   ```bash
   npm install
   ```
